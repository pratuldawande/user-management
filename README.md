# React Native Project

A React Native mobile application for managing users with role-based filtering (All, Admin, Manager).

---

## 📱 Features

* User list screen
* Role based filter tabs (All / Admin / Manager)
* Smooth animated tab slider
* React Navigation stack navigation
* Modern React Native components (Pressable, Animated)

---

## 🛠 Tech Stack

* React Native
* TypeScript
* React Navigation
* Animated API
* sqlite sqlite-storage
* aws-amplify/api

---


## 🚀 Getting Started

### 1. Install dependencies

```
npm install
```

or

```
yarn install
```

---

### 2. Install iOS pods

```
cd ios
bundle install
bundle exec pod install
cd ..
```

---

### 3. Run the application

#### Android

```
npx react-native run-android
```

#### iOS

```
npx react-native run-ios
```

---

## 🧪 Running Tests

```
npm test
```
